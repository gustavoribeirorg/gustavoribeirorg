Migrei o blog para o Github Pages

Já passei por algumas hospedagens e estava utilizando o Cloudflare Pages até agora.

Pra quem acessa, não faz diferença nenhuma. Pra mim, não muda quase também também. Continuo publicando da mesma forma, mas agora tirei o Cloudflare da jogada.

Eu subia os arquivos para o Github e o Cloudflare importava de lá para gerar as páginas. Agora, o Github exibe as páginas diretamente sem um intermediário.

Outro motivo foi que o Cloudflare adicionava um JavaScript de análise de acessos nas páginas e eu não consegui tirar.
